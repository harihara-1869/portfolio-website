# Portfolio Website

This is my personal portfolio website, built using **HTML**, **CSS**, and a little bit of **JavaScript**.  
It’s the first website I’ve created, made to learn the basics of web design and frontend development.

## Features
- Fully responsive layout  
- Clean and minimal UI  
- Simple animations and interactions  
- Sections for Home, About, Portfolio, and Contact  

## Tech Stack
- **HTML**
- **CSS**
- **JavaScript**
